SMS Spam Detector using LLMs

Abstract

Spam messages are a common problem in the modern world. They are not only annoying but also dangerous, as they can be used to steal personal information or spread malware. This project, aims to create a spam detector for SMS messages using Large Language Models (LLMs). Several models are trained on a dataset of SMS messages which then can be used to classify messages as spam or not spam. The goal is to create a spam detector that is accurate, fast, and easy to use.
